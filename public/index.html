Usage Guide

1. API Endpoint

http://127.0.0.1:8765/proxy/v1/chat/completions

2. Python Example

from openai import OpenAI

client = OpenAI(
    base_url="http://127.0.0.1:8765/proxy/v1",
    api_key="any-key"  # Proxy will auto-replace
)

response = client.chat.completions.create(
    model="claude-sonnet-4",
    messages=[{"role": "user", "content": "Hello!"}]
)
print(response.choices[0].message.content)

3. cURL Example

curl http://127.0.0.1:8765/proxy/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer any-key" \
  -d '{
    "model": "claude-sonnet-4",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'

4. How It Works

1. Client sends request to proxy endpoint

2. Proxy selects an available Coding Plan

3. Request is forwarded to target API (using Plan's API Key)

4. If rate limited, automatically switches to next Plan

5. Returns 503 error when all Plans fail

5. Response Headers

X-Failover-Plan - Plan ID used

X-Failover-Attempts - Number of attempts

6. Model Auto-Switch

When a Plan has a model configured, the proxy will automatically replace the model name in requests:

  • Plan with model: Requests will use the Plan's model name
  • Plan without model: Requests will use the client's original model name
  • Use case: Configure different models for different providers (e.g., claude-sonnet-4 → deepseek-chat)

7. Environment Variables

# Linux/macOS
export ANTHROPIC_BASE_URL="http://127.0.0.1:8765/proxy/v1"
export ANTHROPIC_API_KEY="any-key"

export OPENAI_API_BASE="http://127.0.0.1:8765/proxy/v1"
export OPENAI_API_KEY="any-key"

# Windows (PowerShell)
$env:ANTHROPIC_BASE_URL="http://127.0.0.1:8765/proxy/v1"
$env:ANTHROPIC_API_KEY="any-key"
⚠️ This service only listens on 127.0.0.1 by default, for local development only. Configure proper authentication and network isolation for production use.

📜 Universal Startup Script ▼

Linux/macOS: start-with-failover.sh

#!/bin/bash

# Plan Switcher Config
export ANTHROPIC_BASE_URL="http://127.0.0.1:8765/proxy/v1"
export ANTHROPIC_API_KEY="any-key"
export OPENAI_API_BASE="http://127.0.0.1:8765/proxy/v1"
export OPENAI_BASE_URL="http://127.0.0.1:8765/proxy/v1"
export OPENAI_API_KEY="any-key"

echo "🚀 Plan Switcher configured"
echo "   Proxy URL: http://127.0.0.1:8765"
echo ""

# Start tool
exec "$@"

Usage:

chmod +x start-with-failover.sh
./start-with-failover.sh claude
./start-with-failover.sh aider --model claude-sonnet-4
